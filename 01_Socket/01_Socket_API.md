# Socket基础
## API

> 前端

1. socket 建立一个<套接字>
2. connect 连接服务器
3. send 向服务器发送数据
4. recv 接受服务器发送数据
5. closesocket 关闭socket 

> 后端

1. socket 建立一个socket
2. bind 绑定端口
3. listen 监听网络端口
4. accept 等待客户端连接
5. recv 接受客户端数据
6. send 向客户端发送数据
7. closesocket 关闭socket
